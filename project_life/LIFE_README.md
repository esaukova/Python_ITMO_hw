# Игра Жизнь Конвея

Реализация игры "Жизнь" Конвея с поддержкой возраста ячеек и генерацией PNG изображений.

## Установка зависимостей

Для работы программы требуется библиотека Pillow:

```bash
pip install Pillow
```

## Формат входного файла

Входной файл может иметь следующий формат:

1. **Опционально**: первая строка с размерами поля в формате `ширина высота`
2. Далее: строки поля, где:
   - `X`, `1`, `*` - живая ячейка
   - `.`, `0`, пробел - мертвая ячейка
   - Строки, начинающиеся с `#` - комментарии (игнорируются)
   - Пустые строки игнорируются

### Пример входного файла:

```
20 20
....................
........XXX.........
....................
```

или

```
XXX
X.X
XXX
```

## Использование

```bash
python life.py <входной_файл> <выходной_файл> <количество_шагов> [опции]
```

### Параметры:

- `входной_файл` - файл с начальной конфигурацией
- `выходной_файл` - файл для записи всех конфигураций
- `количество_шагов` - количество шагов моделирования

### Опции:

- `--color COLOR` - базовый цвет живой ячейки (по умолчанию: `green`)
  - Доступные цвета: `red`, `green`, `blue`, `yellow`, `cyan`, `magenta`, `white`, `orange`, `purple`
  - Или hex формат: `#FF0000`
- `--cell-size SIZE` - размер ячейки в пикселях (по умолчанию: 10)
- `--png-prefix PREFIX` - префикс для PNG файлов (по умолчанию: `life_step`)

### Примеры:

```bash
# Базовый запуск
python life.py life_input.txt output.txt 10

# С красным цветом и большими ячейками
python life.py life_input.txt output.txt 20 --color red --cell-size 15

# С кастомным префиксом для PNG
python life.py life_input.txt output.txt 5 --png-prefix my_life --color blue
```

## Выходные файлы

1. **Текстовый файл** (`output.txt`): содержит все конфигурации поля на каждом шаге
2. **PNG файлы**: для каждого шага создается отдельный PNG файл:
   - `life_step_000.png` - начальное состояние
   - `life_step_001.png` - после 1 шага
   - `life_step_002.png` - после 2 шагов
   - и т.д.

## Особенности

- **Возраст ячеек**: чем старше ячейка, тем светлее оттенок базового цвета
- **Визуализация**: PNG изображения показывают состояние поля с цветовой индикацией возраста
- **Гибкость**: поддерживаются различные форматы входных файлов

## Правила игры Жизнь

1. Живая ячейка с 2 или 3 соседями выживает
2. Живая ячейка с менее чем 2 соседями умирает (недостаток)
3. Живая ячейка с более чем 3 соседями умирает (перенаселение)
4. Мертвая ячейка с ровно 3 соседями оживает
